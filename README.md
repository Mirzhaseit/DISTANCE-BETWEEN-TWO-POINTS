# DISTANCE-BETWEEN-TWO-POINTS

## AIM:
To write a python program to find the distance two 2 points
## ALGORITHM:
### Step 1: 
import math library
### Step 2: 
using function pass parameters value
### Step 3: 
Substitute the values in the distance formula  ![formula](/formula.JPG)
### Step 4:
return function and call the function by passing arguments
### Step 5:
print the result and end the program
### PROGRAM:
```
# Developed by:MIRZHA FATHIMA S
# Register no:212224230159

import math 
l1=[4,2]
l2=[10,6]
d=math.sqrt((l2[0]-l1[0])**2+(l2[1]-l1[1])**2)
print(f"{d:.2f}")
```
